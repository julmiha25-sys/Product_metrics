Расчет rolling retention.
Интересуют конкретные N-дни: 0, 1, 3, 7, 14, 30, 60 и 90 день.
Пользователей разбиваем по когортам, в качестве признака когорты используем месяц регистрации пользователя.
В ячейках таблицы указано отношение количества людей, которые пришли в день N (или любой последующий) и в день 0 (или любой последующий).
